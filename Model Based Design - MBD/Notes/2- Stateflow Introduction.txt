. Stateflow is a Simulink tool for designing state machines and control logic.

. Create a Stateflow Chart:
    1) Add Chart:
        > Library Browser -> Stateflow -> Chart

    2) Action Language:
        > Model Explorer -> Chart -> Action Language

    3) Add inputs, outputs, variables:
        > Model Explorer -> Chart -> Add Data

    4) Add states and transitions:
        > Enter the Chart -> State

    5) Animation speed
